# NBA Win Probability Streaming ML

## Project Overview
This project is a local Python application that predicts the live, in-game win probability of an NBA game. It utilizes a **Data Stream Mining** approach to process a large Play-by-Play (PBP) dataset incrementally. By avoiding loading the entire dataset into memory at once, the application efficiently handles large volumes of data (millions of rows) in a memory-constrained environment.

The final output is an interactive **Streamlit** dashboard that blends two types of probabilities to calculate real-time predictions:
1. **Historical Deterministic Score:** Based on the results of the 5 most recent matchups between the two selected teams.
2. **Live Probabilistic Score:** Generated by an online streaming machine learning model (`HoeffdingTreeClassifier` from the `river` library) processing real-time game conditions.

These two scores are combined using a time-decaying Prediction Contribution Weight (PCW) as the game progresses.

## Tech Stack
- **Language:** Python 3.9+
- **Database:** SQLite3 (Built-in)
- **Data Processing:** `pandas`, `numpy`
- **Streaming ML:** `river` (specifically `river.tree.HoeffdingTreeClassifier`)
- **Frontend / UI:** `streamlit`, `plotly`

## Concepts & Mathematical Foundations

### 1. Time Remaining Calculation
A standard NBA game is 48 minutes (2880 seconds). The total seconds remaining ($t$) is dynamically calculated based on user input (Quarter, Minutes, Seconds):
$$t = ((4 - Quarter) \times 720) + (Minutes \times 60) + Seconds$$
*(Note: Maximum $t$ is capped at 2880, and minimum at 0. Overtime logic is ignored for this core MVP).*

### 2. The Deterministic Score (Historical Baseline)
Calculates the historical probability of the home team winning based on the last 5 matchups. It uses an exponentially decaying average to give more weight to recent games:
- Let $i$ be the game index ($i=1$ is most recent, $i=5$ is oldest).
- Let $O_i$ be 1 if the Home Team won, and 0 if they lost.

$$Det_{Home} = \frac{\sum_{i=1}^{5} (e^{-i} \cdot O_i)}{\sum_{i=1}^{5} e^{-i}}$$
$$Det_{Away} = 1 - Det_{Home}$$

### 3. The Probabilistic Score (Live PBP)
A `HoeffdingTreeClassifier` streaming model is trained on Play-by-Play data to output the live prediction ($PBP_{Home}$).
- **Features:** `{'time_left': t, 'point_diff': point_differential}`
- **Target:** `home_win` (1 or 0)

### 4. Blending Function (Prediction Contribution Weight - PCW)
The weight of the deterministic score decays linearly as time ticks down from 2880 to 0. 
$$PCW = \frac{t}{2880}$$

The final probabilities are blended seamlessly:
$$Final_{Home} = (PCW \cdot Det_{Home}) + ((1 - PCW) \cdot PBP_{Home})$$
$$Final_{Away} = 1 - Final_{Home}$$

## Local Setup & Installation

### Prerequisites
- Python 3.9 or higher
- `pip` package manager

### 1. Clone the repository
```bash
git clone https://github.com/varun4sid/data-mining-project.git
cd data-mining-project
```

### 2. Create a Virtual Environment & Install Dependencies
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

### 3. Database Setup & Data Ingestion
This script creates the `nba_data.db` SQLite database and ingests the dataset files (`game_summary.csv` and `play_by_play.csv`) incrementally to preserve memory.
```bash
python db/db_setup.py
```

### 4. Train the Streaming ML Model
Train the `HoeffdingTreeClassifier` directly from the SQLite database stream. This will output a serialized `hoeffding_model.pkl` file.
```bash
python src/train_model.py
```

### 5. Run the Streamlit Dashboard
Launch the interactive UI to predict live probabilities.
```bash
streamlit run src/app.py
```
